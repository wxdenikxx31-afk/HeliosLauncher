# Changelog — UniCraft Launcher

## [2.4.0] - 2026-04-19 — Официальный релиз!

### 🚀 Переименование
- Лаунчер переименован из **Helios Launcher** в **UniCraft Launcher**
- Обновлены все ссылки, заголовки, описания и внутренние идентификаторы
- Новый appId: `unicraftlauncher`
- Новое имя установщика: `UniCraft Launcher-setup-VERSION.exe`

### 🔄 Миграция данных
- Автоматическая миграция данных из `.helioslauncher` → `.unicraftlauncher`
- При обновлении с v2.3.x все данные переносятся без потерь:
  - Java, моды, конфиги, настройки, аккаунты
- Путь `dataDirectory` в config.json обновляется автоматически

### 🛡️ Сохранение данных
- Подтверждено: обновления **не удаляют** Java, моды и конфиги
- Кастомный NSIS-скрипт удаляет только файлы приложения

### 📝 Прочее
- Удалён `.github/FUNDING.yml` (донаты оригинального автора)
- README.md полностью переписан для UniCraft
- Добавлен кредит оригинальному проекту Helios Launcher

## [2.3.6–2.3.7] - 2026-04-18

### Исправлено
- **Критическое исправление**: данные пользователя (Java, моды, конфиги) больше не удаляются при обновлении
- Кастомный `installer.nsh` — NSIS удаляет только файлы приложения (exe, dll, resources/, locales/), все остальные директории сохраняются
- `config.json` перемещён в `%APPDATA%\.helioslauncher\` — безопасная директория, не затрагиваемая установщиком

## [2.3.0] - 2026-04-17

### Добавлено
- **Автообновление через GitHub** — лаунчер проверяет обновления при запуске и показывает полноэкранный оверлей с прогресс-баром загрузки.
- **Вкладка «Консоль и Dev Tools»** в настройках — просмотр логов лаунчера и игры в реальном времени с фильтрами по уровню (INFO, WARN, ERROR, DEBUG), поиском, экспортом в файл и копированием в буфер обмена.
- **Диагностика системы** — карточки с информацией об ОС, CPU, RAM, GPU, Java, версии лаунчера и Electron прямо во вкладке Dev Tools.
- **Кнопка консоли в заголовке окна** — иконка терминала рядом с кнопками управления окном для быстрого доступа к Dev Tools.
- **Горячая клавиша Ctrl+Shift+C** — открывает вкладку консоли из любого экрана лаунчера.
- **Перехват логов игры** — stdout/stderr процесса Minecraft перенаправляются в консоль лаунчера.

### Изменено
- **Увеличены размеры шрифтов** — все шрифты увеличены на 2px для лучшей читаемости.
- **IP сервера** обновлён на `d10.aurorix.net:25257`.
- **Публикация обновлений** настроена через GitHub-репозиторий `wxdenikxx31-afk/HeliosLauncher`.
